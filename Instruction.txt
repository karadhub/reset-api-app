🧩 Task 1 – CI/CD + Containerization (45 mins)
Goal: See how he automates build & deploy.
Scenario:
 Mahad IT runs a small Node.js API service that must be containerized and automatically deployed on each Git push.
Instructions:
1.	Create a small Node.js “Hello Mahad” API.

2.	Write a Dockerfile to containerize it.

3.	Create a GitHub Actions workflow (or Jenkinsfile) that:

○	Builds and pushes the Docker image to Docker Hub (or GHCR).

○	Runs the container automatically (locally or on minikube).

4.	Add a simple README with run instructions.
